# GigShield AI

AI-powered parametric insurance platform for food delivery partners.

Target Users:
Swiggy and Zomato delivery partners.

Problem:
Delivery workers lose income due to heavy rain, floods, extreme heat and pollution.

Solution:
GigShield provides AI-based weekly insurance coverage that automatically compensates workers when disruptions occur.

Key Features
- Worker onboarding
- AI-based weekly premium calculation
- Weather based disruption triggers
- Automatic claim approval
- Fraud detection
- Instant payout simulation

Technology Stack
Frontend: HTML CSS JavaScript
Backend: Python Flask
AI Models: Python
APIs: Weather API
